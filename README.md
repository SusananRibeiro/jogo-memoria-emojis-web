# Jogo da Memória de Emojis

Este é um jogo da memória que utiliza o teclado de emojis do Windows como tema. O objetivo é combinar pares de emojis idênticos, virando-os de cabeça para baixo, até que todos os pares sejam encontrados. Este projeto é desenvolvido utilizando HTML, CSS e JavaScript, enfatizando a estilização CSS e manipulação de eventos JavaScript.

## Tecnologias Utilizadas

- JavaScript
- HTML
- CSS
- Visual Studio Code

## Nível do Projeto

Este projeto é classificado como Front-End: Intermediário. É adequado para aqueles com conhecimento básico em HTML, CSS e JavaScript, mas que desejam aprimorar suas habilidades de estilização e manipulação de eventos em JavaScript.

## Como Jogar

1. Abra o arquivo `index.html` em seu navegador da web.
2. O tabuleiro do jogo será exibido emojis virados de cabeça para baixo.
3. Clique em duas cartas para virá-las.
4. Se os emojis forem idênticos, eles permanecerão virados para cima.
5. Se não forem, eles serão virados de cabeça para baixo novamente.
6. Continue virando cartas até encontrar todos os pares de emojis idênticos.

## Recursos Adicionais

Além do jogo principal, este projeto também inclui:

- Estilização avançada utilizando CSS para criar uma interface atraente e responsiva.
- Manipulação de eventos JavaScript para controlar a lógica do jogo, incluindo virar cartas e verificar pares.
